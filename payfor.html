미납요금 납부 방법은 금융기관납부 / 영업소 사무실 방문 납부 총 2가지가 있습니다!

미납통행료 전화 문의

직접 계좌이체

금융기관을 방문하여 계좌이체

인터넷 납부 인터넷 뱅킹

영업소/사무실 방문납부

영업소 방문 납부

광주제2순환도로 소태·소암 영업소를 방문하여
요금소 현장에서는 현금으로만 납부가 가능합니다.

사무실 방문 납부

주제2순환도로 소태·소암 영업소 사무실에서는
현금 / 선·후불 전자카드 / 선·후불 교통카드로 납부가 가능합니다.

※선불카드사 : 하이플러스 / 한국도로공사 / 티머니 / 한페이시스 / 캐시피(EB) / 레일플러스 / 유페이먼트
※후불카드사(신용) : 국민 / 비씨 / 현대 /삼성 / 신한 / 수협 / 롯데 / 외환-하나 / 농협 / 광주 / 전북 / 씨티

당일 미납 통행료 납부 안내
차로(요금소)에서는 당일 발생한 미납요금 확인은 불가합니다.
수납원에게 미리 말씀해 주시면 확인 및 납부가 가능하오니 협조 부탁드립니다. 이용에 불편을 드려 대단히 죄송합니다.
